命令之后要打空格加符号

1.is命令：是list的缩写，可以看到linux文件夹包含的文件，并且可以看到文件权限，目录信息等eg：ls-a
累出目录所有的文件，包含以.开始的隐藏文件。
-A累出.及..的其他文件
-r反序排列
-t 以文件大小排序
-S 以文件大小排序
-h 以易读大小显示
-l除了文件名之外，还将文件的权限、所有者、文件大小等信息详细类出来

2.cd命令：
cd/ 进入要目录
cd~进入xxx目录
cd-进入上一次工作路径
cd ！￥把上个命令的参数作为cd参数使用

3.pwd命令：命令用于查看当前工作目录路径
（1）查看当前路径 pwd
（2）pwd -p 查看软链接的实际路径

4.mkdir命令：
用于创造文件夹：
-m 对新建目录设置存根权限，也可以用chmod命令设置：
-p 可以是一个路径名称。此时若路径中的某些目录上不存在，加上此选项后，系统将自动建立
好那些尚不存在的目录，记一次可多个目录

mkdir t 可以在当前目录下创建为t的文件夹mkdir -p /tmp/test/t1/t

mkdir t 生成的文件带锁，所以需要chmod +777 t 来将他的所给去掉

5.rmdir命令：

可以删除空目录，不可删非空目录

6.rm

rm -i *.log 删除任何一个.log文件，删除前逐一询问确认： rm -i *.log

rm -rf test 删除test子目录及子目录中所有档案删除，并且不用一一确认。

删除以rm – -f*开头的文件，不能是目录

注：[rm与rmdir有什么区别_百度知道 (baidu.com)](https://zhidao.baidu.com/question/115911458.html#:~:text=rmdir命令是用来删除空的目录，rm命令是用来删除非空的目录。 Linux中rmdir命令是用来删除空的目录。 参数：,-p 是当子目录被删除后使它也成为空目录的话，则顺便一并删除。 删除名字为folder1的子目录。 如果想删除非空目录，要使用rm才可以。)

7.mv

mv test.log test1.txt :将test.log改为test1.txt

mv -i log1.txt log2.txt log3.txt ：将文件log1.txt 2 3 移动到根的test3目录中

mv -i log1.txt log2.txt 将文件

mv * ../不是很理解。。。

cp -i 提示

cp -r复制目录及目录内所有项目

cp -a 复制的文件与原文件时间一样???

9.cat命令

一次显示整个文件：cat filename

从键盘创造一个文件：cat > filename
